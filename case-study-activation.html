20% Activation Lift Through Personalized User Onboarding

The Goal: Improve usage retention by finding the highest leverage intervention point.

What were the challenges?

Find high leverage actions to improve usage retention. Overall retention was stable at ~70% MoM, but through segmentation analysis I identified an uplift opportunity with new users: only 45% returned in their second month.

Diving deeper into new users experience and their retention, I identified this pattern: new users who used "advanced high value features" were 2x more likely to return (2.5x if they repeated usage). And this agrees with common sense

Key insight: Activation is a proxy for 'value provided fast' and strongly correlates with retention. The question: can we guide users to these features without overwhelming them? Will it help those new users find more value, and faster?

How did I contribute to address this?

My recommendation: Make it easier for new users to discover relevant advanced features for their use case.

The PM designed an onboarding survey solution. I collaborated on the experiment design and analysis, structuring it as an A/B test with three groups to isolate both checklist and personalization effects.

The Experiment

We tested two hypotheses:

  • Checklist effect: Does providing recommended actions improve activation?
  • Personalization effect: Does asking about use case and personalizing recommendations add value?

Appendix: Detailed Experiment Results

Checklist Effect

Hypothesis: Does providing recommended actions improve activation?

  • Result: 50% activation lift (23% control vs 35% treatment)
  • Users in Treatment: 827 (60%)
  • Users in Control: 537

Personalization Effect

Hypothesis: Does personalization add value beyond generic checklist?

  • Result: ~50% additional lift vs default checklist (30% vs 43%)
  • Combined effect: 119% lift vs control (20% vs 43%)
  • Users in Treatment (Fundraising): 310 (37%)
  • Users in Control (Not fundraising): 517

Statistical Methodology

  • Welch t test (robust to distributional assumptions)
  • Group equivalence validated (no pre-existing differences)
  • Statistical significance: alpha = 5%
  • Experiment duration: Several weeks for sufficient sample size
